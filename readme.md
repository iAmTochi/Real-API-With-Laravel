# Real World Restful Api using Laravel

## Installation
* Clone Repo
* Install packages

```bash
composer install
```

* Duplicate `.env.example`

```bash
cp .env.example .env
```

* Update database credentials

* Generate app key

```bash
php artisan key:generate
```

* Run Migrations

```bash
php artisan migrate --seed
```

* Install Passport

```bash
php artisan passport:install
```
